## Overview

You can build content switcher by using a combination of the `ContentSwitcher` and `Switch` components. The `ContentSwitcher` components accepts a list of `Switch` components as children and each `Switch` is responsible for displaying the section heading.

You can configure the `Switch` heading using the `text` prop.

```jsx
import { ContentSwitcher, Switch } from "carbon-components-react";

function MyComponent() {
  return (
    <ContentSwitcher>
      <Switch text="First section" />
      <Switch text="Second section" />
      <Switch text="Third section" />
    </ContentSwitcher>
  );
}
```

## Component API

### ContentSwitcher props

| Prop          | Type                         | Required | Default     | Description                                                                                              |
| ------------- | ---------------------------- | -------- | ----------- | -------------------------------------------------------------------------------------------------------- |
| selectedIndex | `number`                     | –        | 0           | Specify a selected index for the initially selected content                                              |
| selectionMode | oneOf 'automatic' / 'manual' | –        | 'automatic' | Choose whether or not to automatically change selection on focus                                         |
| children      | `node`                       | `true`   | –           | Pass in Switch components to be rendered in the ContentSwitcher                                          |
| className     | `string`                     | –        | –           | Specify an optional className to be added to the container node                                          |
| onChange      | `func`                       | 'true'   | –           | Specify an `onChange` handler that is called whenever the ContentSwitcher changes which item is selected |

### ContentSwitcher selectedIndex

You can control which item is selected for the content switcher by using the `selectedIndex` prop.

```jsx
<ContentSwitcher selectedIndex={1}>
  <Switch text="First section" />
  <Switch text="Second section" />
  <Switch text="Third section" />
</ContentSwitcher>
```

### ContentSwitcher selectionMode

The `selectionMode` prop can be used to allow a user to change focus without changing the selection by passing in `manual`.

```jsx
<ContentSwitcher selectionMode="manual">
  <Switch text="First section" />
  <Switch text="Second section" />
  <Switch text="Third section" />
</ContentSwitcher>
```

### Switch props

| Prop      | Type                | Required | Default        | Description                                                                                                             |
| --------- | ------------------- | -------- | -------------- | ----------------------------------------------------------------------------------------------------------------------- |
| selected  | `bool`              | –        | `false`        | Whether your Switch is selected. Reserved for usage in ContentSwitcher                                                  |
| text      | `string`            | `true`   | 'Provide text' | Provide the label of your Switch                                                                                        |
| onClick   | `func`              | –        | –              | A handler that is invoked when a user clicks on the control. Reserved for usage in ContentSwitcher                      |
| onKeyDown | `func`              | –        | –              | A handler that is invoked on the key down event for the control. Reserved for usage in ContentSwitcher                  |
| className | `string`            | –        | –              | Specify an optional className to be added to your Switch                                                                |
| index     | `number`            | –        | –              | The index of your Switch in your ContentSwitcher that is used for event handlers. Reserved for usage in ContentSwitcher |
| name      | 'string' / 'number' | –        | –              | Provide the name of your Switch that is used for event handlers                                                         |

### Switch selected

The `selected` is used to specify weather the specific `Switch` is selected.

```jsx
<ContentSwitcher selectionMode="manual">
  <Switch selected={false} text="First section" />
  <Switch selected={false} text="Second section" />
  <Switch selected={false} text="Third section" />
</ContentSwitcher>
```

## Accessibility considerations

Each `Switch` component itself is rendered as a `button` and is considered an interactive element. If you render an interactive element inside of this component, then it becomes nested inside of the `button`. Common examples of this are links. Rendering interactive content inside of the `Switch` is not reachable via a keyboard or screen reader, so we do not recommend this.
