## Overview

You can build content switcher by using a combination of the `ContentSwitcher` and `Switch` components. The `ContentSwitcher` components accepts a list of `Switch` components as children and each `Switch` is responsible for displaying the section heading.

You can configure the `Switch` heading using the `text` prop.

```jsx
import { ContentSwitcher, Switch } from "carbon-components-react";

function MyComponent() {
  return (
    <ContentSwitcher onChange={function noRefCheck() {}}>
      <Switch
        name="one"
        onClick={function noRefCheck() {}}
        onKeyDown={function noRefCheck() {}}
        text="First section"
      />
      <Switch
        name="two"
        onClick={function noRefCheck() {}}
        onKeyDown={function noRefCheck() {}}
        text="Second section"
      />
      <Switch
        name="three"
        onClick={function noRefCheck() {}}
        onKeyDown={function noRefCheck() {}}
        text="Third section"
      />
    </ContentSwitcher>
  );
}
```
